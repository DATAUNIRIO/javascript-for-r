JavaScript for R

Chapter 19 Rmarkdown

Rmarkdown (Allaire et al. 2020) is an R package that allows the creation of documents from R, these range from presentations with xaringan (Xie 2020b), through dashboard with flexdashboard (Iannone, Allaire, and Borges 2020) to entire books such as this one with bookdown (Xie 2020a).

Many of these outputs being interactive HTML documents they allow using including JavaScript.

## Examples

Packages that make use of JavaScript to extend rmarkdown are numerous and very different in nature given the variety of outputs that can be produced.
